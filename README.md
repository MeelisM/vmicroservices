## Application

### inventory-app

Start the server.

```
npm run start
```

Run database migrations.

```
npm run db:migrate
```

Roll back the last database migration.

```
npm run db:migrate:undo
```

Roll back all database migrations.

```
npm run db:migrate:undo:all
```
